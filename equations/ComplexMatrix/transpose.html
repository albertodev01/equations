ComplexMatrix transpose()
override

Returns the transpose of this matrix.

This is an operation that flips a matrix over its diagonal (switching the row and column indices of the matrix to create a new one). The transpose of an N×M matrix is an M×N matrix where the element at position (i, j) in the original matrix becomes the element at position (j, i) in the transposed matrix.

Implementation

@override
ComplexMatrix transpose() {
  final source = List<Complex>.generate(
    rowCount * columnCount,
    (_) => const Complex.zero(),
  );

  for (var i = 0; i < rowCount; i++) {
    for (var j = 0; j < columnCount; j++) {
      source[rowCount * j + i] = this(i, j);
    }
  }

  return ComplexMatrix.fromFlattenedData(
    rows: columnCount,
    columns: rowCount,
    data: source,
  );
}
