PolynomialLongDivision

PolynomialLongDivision class

Implements the polynomial long division algorithm, which divides one polynomial by another polynomial.

This implementation requires that the degree of the denominator be less than or equal to the degree of the numerator. If this condition is not met, a PolynomialLongDivisionException will be thrown.

Algorithm Overview

The polynomial long division algorithm works similarly to numerical long division:

  1. Divide the leading term of the numerator by the leading term of the denominator
  2. Multiply the entire denominator by this quotient
  3. Subtract the result from the numerator
  4. Repeat until the degree of the remainder is less than the degree of the denominator

Example

final division = PolynomialLongDivision(
  polyNumerator: Algebraic.fromReal([1, 3, -6]), // P(x) = x^2 + 3x - 6
  polyDenominator: Algebraic.fromReal([1, 7]),   // Q(x) = x + 7
);

final result = division.divide();
// result.quotient = x - 4
// result.remainder = 22
