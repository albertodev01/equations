Riddler

Riddler class final

Implements Riddler's method (also known as the Ridders' method) to find the roots of a given equation.

Uses exponential interpolation to find the root. It's a modification of the false position method that improves convergence.

  • The method requires the root to be bracketed between two points a and b (i.e., f(a) * f(b) < 0), otherwise it won't work.

  • The rate of convergence is approximately sqrt(2) (superlinear), which is faster than bisection but slower than Newton's method.

  • The convergence is guaranteed for continuous functions when the root is properly bracketed, making it more reliable than methods like Newton's method for ill-behaved functions.

  • The algorithm uses the formula: x = x2 + (x2 - x0) * sign(y0 - y1) * y2 / sqrt(y2^2 - y0 * y1) where x2 is the midpoint and y0, y1, y2 are function values.

  • The method may fail if the expression under the square root becomes negative or zero, in which case the algorithm falls back to bisection.
